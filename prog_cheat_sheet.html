Python Boolean Operators and Django ORM Field Lookups

Python Operator Description Django lookup / pattern Example
and / or / not (logical) Combine query conditions Use Q objects combined with &, |, ~ from django.db.models import Q
Book.objects.filter(Q(rating__gte=4) & Q(is_bestselling=True))
== Equality field__exact or field__iexact Book.objects.filter(title__exact='Dune')
!= Inequality Use .exclude() or ~Q(...) Book.objects.exclude(title__exact='Dune')
>, >=, <, <= Comparisons field__gt, field__gte, field__lt, field__lte Book.objects.filter(rating__gte=4)
in Membership in a list/sequence field__in Book.objects.filter(id__in=[1,2,3])
is None / is not None Check for NULL field__isnull=True/False Book.objects.filter(author__isnull=True)
in (substring) Substring membership field__contains, field__icontains Book.objects.filter(title__icontains='python')
startswith / endswith Prefix/suffix match field__startswith, field__istartswith, field__endswith, field__iendswith Book.objects.filter(title__istartswith='The')
regex Regular expression match field__regex, field__iregex Book.objects.filter(title__iregex=r"^The\\s")
range Between two values field__range Book.objects.filter(rating__range=(2,4))

Bulk Django ORM Operations

Operation What it does Method / Example Docs
Delete multiple Delete all records in a queryset at once. Book.objects.filter(rating__lt=2).delete() Deleting objects
Update multiple Set a field value on every record in the queryset (runs a single UPDATE SQL). Book.objects.filter(is_bestselling=True).update(rating=5)
Or use bulk_update() for many model instances.		 bulk_update / update
Create multiple Insert many model instances in a single query (more efficient than saving one-by-one). Book.objects.bulk_create([Book(title='A'), Book(title='B')]) bulk_create
